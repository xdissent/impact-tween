Impact Tween Plugin Example

This example uses the Impact Tween Plugin. A tween is created in Weltmeister that eases the game's screen down the Y axis as the character falls (due to gravity). A tween trigger area is set up just above the floor so that as the character lands, another tween is triggered to bounce the other characters into the air. The initial screen tween is started within the game code, but everything is built exclusively in Weltmeister.

To build this example, unpack the lib/game and media folders into the ImpactJS source. Then install the Impact Leveldata Plugin and Impact Tween Plugin as described in the README.
